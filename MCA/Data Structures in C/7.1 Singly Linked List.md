### Key Functions:

1. **Insertion**:
    
    - **FirstNode()**: Creates the first node in the list when it's empty.
    - **InserAtBeginning()**: Inserts a new node at the beginning of the list.
    - **InserAtMiddle()**: Inserts a node at a specific position in the middle of the list.
    - **InserAtEnd()**: Adds a new node at the end of the list.
2. **Deletion**:
    
    - **DelB()**: Deletes the first node (beginning of the list).
    - **DelM()**: Deletes a node at a specified position.
    - **DelE()**: Deletes the last node (end of the list).
    - **DelFirst()**: Helper function to delete the first node when there's only one node.
3. **Utility**:
    
    - **Display()**: Displays all elements of the list, along with the count.
    - **Search()**: Searches for a specific element in the list and reports its position.
    - **isEmpty()**: Checks if the list is empty.
    - **Length()**: Returns the current length of the list.
    - **LastNode()**: Returns the last node of the list to facilitate inserting at the end.
