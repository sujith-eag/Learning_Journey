`\t` horizontal tab space
`\v` vertical tab space
`\n` new line

`\a` bell or alert
`\b` back space
`\r` enter button
`\\` single `\`
`\+` for `+`

`find  rfind  index  rindex`

replacing string with another
`string.replace(old, new)`
